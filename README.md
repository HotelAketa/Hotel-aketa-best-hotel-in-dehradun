# Hotel Aketa Dehradun - Official Website

A fully responsive, SEO-optimized single-page landing website for Hotel Aketa Dehradun, showcasing the premier business and family hotel on Rajpur Road.

## Project Overview

This website presents Hotel Aketa as the best hotel in Dehradun, featuring modern amenities, comfortable accommodations, banquet facilities, and exceptional service for both business and leisure travelers.

## Features

### Design & User Experience
- **Fully Responsive**: Mobile-first design that works seamlessly across all devices
- **Modern UI**: Clean, professional layout with Tailwind CSS
- **Smooth Scrolling**: Internal navigation with smooth scroll behavior
- **Interactive Elements**: Hover effects, transitions, and card animations
- **Sticky Navigation**: Easy access to all sections from anywhere on the page

### Sections
1. **Hero Section**: Eye-catching header with clear call-to-action buttons
2. **About**: Introduction to Hotel Aketa and its offerings
3. **Amenities**: Showcase of world-class facilities with icon-based cards
4. **Rooms**: Display of different room types with features
5. **Banquet**: Information about event and conference facilities
6. **Contact**: Contact information and inquiry form
7. **Footer**: Quick links and contact details

### SEO Optimization
- **Semantic HTML5**: Proper structure with header, main, section, nav, footer tags
- **Meta Tags**: Comprehensive meta descriptions and keywords
- **Open Graph**: Social media sharing optimization
- **Twitter Cards**: Enhanced Twitter sharing
- **JSON-LD Schema**: LocalBusiness structured data for rich snippets
- **Canonical URL**: Proper canonical URL specification
- **Optimized Headings**: H1 and H2 tags with target keywords:
  - Best Hotel in Dehradun
  - Top Hotel on Rajpur Road Dehradun
  - Business Hotel in Dehradun
  - Family Hotel in Dehradun
- **robots.txt**: Search engine crawling instructions
- **sitemap.xml**: Complete sitemap for search engines

## Technology Stack

- **HTML5**: Semantic markup
- **Tailwind CSS**: Utility-first CSS framework (via CDN)
- **Vanilla JavaScript**: Minimal JS for navigation and smooth scrolling
- **Vite**: Modern build tool and dev server

## Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository or extract the files

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

5. Preview production build:
```bash
npm run preview
```

## Deployment

### GitHub Pages
This project is ready for GitHub Pages deployment:

1. Build the project:
```bash
npm run build
```

2. The `dist` folder contains the production-ready files

3. Push to GitHub and enable GitHub Pages in repository settings

### Other Hosting Platforms
The built files in the `dist` folder can be deployed to any static hosting service:
- Netlify
- Vercel
- AWS S3
- Cloudflare Pages
- Firebase Hosting

## Customization

### Update Contact Information
Edit the following in `index.html`:
- Email: `info@hotelaketadehradun.com`
- Phone: Replace `+91-XXXXXXXXXX` with actual number
- Address details if needed

### Change Colors
The color scheme uses Tailwind CSS classes. Main colors:
- Primary: Blue (blue-600, blue-700)
- Background: Gray shades
- Hero gradient: Purple to blue gradient

### Add Images
Replace placeholder gradients with actual images:
1. Add images to the `public/images/` folder
2. Update the `src` attributes in the HTML
3. Update Open Graph image URL in meta tags
